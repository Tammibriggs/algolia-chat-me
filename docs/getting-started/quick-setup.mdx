# Quick Setup

Initialize the ChatMe SDK in your app to start using its features.

## Basic Initialization

```js
const sdk = new ChatMe({
  token: 'YOUR_API_TOKEN',
  userId: 'USER_ID', // Optional, for user tracking
});
```

- `token`: Your API token (required)
- `userId`: Unique user identifier (optional)

## Example Use Case

Use this setup to enable chat, feedback, and more.
