# Installation

Get up and running with the ChatMe SDK in minutes.

## NPM

Install via npm:

```bash
npm install chatme-sdk
```

Import in your app:

```js
import ChatMe from 'chatme-sdk';
```

## CDN

Add the script tag to your HTML:

```html
<script src="https://cdn.chatme.com/sdk/latest.js"></script>
```

The SDK will be available as `window.ChatMe`.
